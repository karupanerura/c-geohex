# c-geohex

GeoHex implement in C99.

# What is GeoHex?

https://sites.google.com/site/geohexdocs/
http://geohex.net/

# Dependencies

* picotest
* cmake
* prove

# Cheat Sheet

## SETUP

```bash
git clone https://github.com/karupanerura/c-geohex.git
cd c-geohex
git submodule update --init
```

## BUILD

```bash
cmake .
make
```

## TEST

```bash
make test
```
